# PHP: Путь от новичка до Junior

## Краткий план (6 месяцев)

- Месяц 1: Основы PHP (синтаксис, переменные, типы, операторы) + простые задачи
- Месяц 2: Функции, массивы, работа со строками, регулярные выражения
- Месяц 3: Веб в PHP — формы, сессии, куки, отправка писем
- Месяц 4: Базы данных (MySQL), PDO, безопасность (SQL-инъекции)
- Месяц 5: ООП в PHP, Composer, автозагрузка, тестирование (PHPUnit)
- Месяц 6: Фреймворк (Laravel или Symfony), деплой, подготовка портфолио

---

## Цель курса
Сформировать набор задач и проектных упражнений, которые последовательно прокачают практические навыки PHP: от базового синтаксиса и алгоритмов до создания небольших веб-приложений и работы с фреймворком. По окончании — вы сможете претендовать на позицию Junior PHP Developer.

# ТЕМЫ И ЗАДАЧИ

Каждая тема содержит описания задач (минимум 10 задач для ключевых областей). У каждой задачи — требования и критерии успешного выполнения.

## ТЕМА 1: Основы PHP (10 задач)

1. Hello, PHP!
   - Создайте файл `index.php`, выведите "Hello, PHP!" и текущую дату/время
   - Используйте короткий и длинный синтаксис для echo/print

2. Переменные и типы
   - Объявите переменные разных типов (string, int, float, bool, null)
   - Используйте `var_dump()` и `gettype()` для вывода типов

3. Приведение типов
   - Покажите примеры приведения типов (int от строки, float от bool и т.д.)
   - Рассмотрите поведение при нечисловых строках

4. Арифметика и операторы
   - Решите простые вычисления, используйте операторы инкремента/декремента
   - Напишите функцию для вычисления факториала

5. Условные конструкции
   - if / else / elseif / тернарный оператор
   - Задача: проверка года на високосность

6. Циклы
   - for, foreach, while, do-while
   - Задача: вывести таблицу умножения, суммировать числа 1..N

7. Работа со строками
   - strlen, substr, strpos, str_replace, mb_* функции
   - Задача: реверс строки, проверка палиндрома (учёт UTF-8)

8. Массивы (обзор)
   - Индексные и ассоциативные массивы
   - array_push, array_pop, array_shift, array_unshift
   - Задача: объединить 2 массива и убрать дубликаты

9. Суперглобальные переменные
   - $_GET, $_POST, $_SERVER, $_FILES
   - Пример: обработать параметры из query string

10. Файловые операции (базовые)
    - fopen, fread, fwrite, file_get_contents, file_put_contents
    - Задача: логирование сообщений в файл с датой и временем

## ТЕМА 2: Функции, массивы и регулярные выражения (10 задач)

1. Функции и область видимости
   - Объявление функций, возвращаемые значения, передачa по ссылке
   - Задача: написать функцию, вычисляющую n-е число Фибоначчи (итер.)

2. Анонимные функции и замыкания
   - Использование `function()` и `use`
   - Задача: фильтрация массива с использованием callback

3. Работа с массивами (продв.)
   - array_map, array_filter, array_reduce, array_column
   - Задача: подсчитать количество вхождений значений в массиве

4. Сортировка массивов
   - sort, rsort, asort, arsort, usort (с пользовательской функцией)
   - Задача: отсортировать массив объектов по нескольким полям

5. Регулярные выражения
   - preg_match, preg_replace, preg_split
   - Задача: валидация email и парсинг телефонных номеров

6. Обработка ошибок и исключений
   - try/catch, Exception, пользовательские исключения
   - Задача: реализация функции деления с обработкой деления на ноль

7. Работа со строками — продвинутый уровень
   - sprintf, vsprintf, printf
   - Задача: форматирование отчёта (табличный вывод текста)

8. Сериализация и JSON
   - serialize, unserialize, json_encode, json_decode
   - Задача: экспорт/импорт данных в JSON-файл

9. Генераторы (Generators)
   - yield, итерация больших наборов данных без загрузки в память
   - Задача: генератор для построчной обработки большого файла

10. Unit тесты для функций
    - Введение в PHPUnit: написать простые тесты для ранее созданных функций
    - Критерий: автотесты должны проходить

## ТЕМА 3: Веб-программирование в PHP (формы, сессии, загрузки) (10 задач)

1. Обработка форм (GET/POST)
   - Создать форму с несколькими полями
   - Обработать ввод: валидация, очистка (sanitize), вывод результата

2. Загрузка файлов
   - Обработка `$_FILES` — ограничения по размеру и типу, защита от злонамеренных файлов
   - Задача: загрузить картинку, сохранить с уникальным именем, сделать thumbnail

3. Сессии и куки
   - session_start, $_SESSION, setcookie
   - Задача: реализовать простую систему аутентификации (login/logout) с сессиями

4. Отправка писем
   - mail() и использование PHPMailer
   - Задача: отправить письмо с подтверждением регистрации (HTML письмо)

5. CSRF защита
   - Что такое CSRF, реализация токенов в формах
   - Задача: защитить форму от CSRF

6. Обработка AJAX-запросов
   - Приём JSON данных и ответ JSON
   - Задача: создать endpoint для поиска (autocomplete) по базе

7. REST-подход в PHP
   - Создать простой REST API: CRUD для сущности "задача"
   - Использовать правильные методы HTTP и статус-коды

8. Маршрутизация без фреймворка
   - Реализовать простой роутер (чтение PATH_INFO и вызов контроллера)
   - Задача: маршруты для index, /tasks, /tasks/123

9. Безопасность ввода
   - XSS, SQL-инъекции — примеры и защита
   - Задача: продемонстрировать XSS и потом исправить (экранизация)

10. Логирование и мониторинг
    - Логирование ошибок, ротация логов
    - Создать простой dashboard с количеством запросов / ошибками

## ТЕМА 4: Базы данных и PDO (10 задач)

1. Подключение к MySQL через PDO
   - Настройка PDO, DSN, опции (ERRMODE_EXCEPTION)
   - Задача: безопасное подключение и обработка ошибок

2. CRUD операции с PDO
   - Подготовленные выражения (prepared statements)
   - Задача: реализовать CRUD для таблицы `users`

3. Транзакции
   - beginTransaction, commit, rollback
   - Задача: реализация перевода баланса между счетами с транзакцией

4. Индексы и оптимизация запросов
   - EXPLAIN, индексы, нормализация/денормализация
   - Задача: проанализировать медленный запрос и предложить индекс

5. Реляционные связи
   - One-to-many, many-to-many — схемы и реализации
   - Задача: модель Orders/OrderItems/Products и запрос получения заказа с элементами

6. Pagination и лимит
   - OFFSET/LIMIT, лучшие практики, keyset pagination
   - Задача: реализовать пагинацию для списка товаров

7. Поиск и полнотекстовый поиск
   - LIKE vs FULLTEXT, использование MATCH...AGAINST
   - Задача: реализовать поиск по названию и описанию

8. Миграции и версии БД
   - Простейшая система миграций (SQL файлы + трекинг)
   - Задача: написать миграции для создания основных таблиц

9. ORM vs PDO
   - Обзор ORM (Eloquent), плюсы и минусы
   - Задача: реализовать простую обёртку-репозиторий поверх PDO

10. Резервное копирование и восстановление
    - Экспорт/импорт данных, dump, импорт больших файлов
    - Задача: скрипт для бэкапа базы и восстановления из дампа

10. PSR стандарты и автозагрузка
